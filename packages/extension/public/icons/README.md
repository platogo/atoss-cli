# Extension Icons

Place the following icon files in this directory:
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

For now, you can use any PNG images or create simple placeholder icons.

To create placeholder icons on macOS:
```bash
# Create a simple colored square as placeholder
convert -size 16x16 xc:#0066cc icon-16.png
convert -size 48x48 xc:#0066cc icon-48.png
convert -size 128x128 xc:#0066cc icon-128.png
```

Or copy any existing PNG and resize it to the required dimensions.
