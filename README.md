# About

A Clojure CLI tool designed to interact with ATOSS. 

## Usage

### Requirements

You need a decently modern version of JRE to run built `atoss-cli` JARs.

For development, you will need:

- Clojure
- Leiningen
- Up to date `chromedriver` and Google Chrome.
